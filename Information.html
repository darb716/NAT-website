What is this website

If you have just stumbled upon this website this will help to give you an explanation of what it's purpose. This website was created to help people learn networking. Specifically it was created to help people better understand now Network Address Translation(NAT), Port Address Translation(PAT), as well as basic packet movement across an extremely basic network.

The home page of this network is basically a FAQ/Sarcastic answer to what this website is

This website was created to be as granular as possible. By that I mean no matter what level of skill you have I hope you can get somthing out of this website. Starting out i would suggest the No Nat page, and move to Simple Nat and Pat from there.

Symbosl/Items that are uncommon

This website uses fairly common images that are used for most network diagrams. However if you do not understand them I will explain them here:

This is a router. Routers are the most important devices for the networks we will be working with on this website. They do all the NAT and all the PAT.

This is a client. Clients can be thought of as any computer that tries to go to go out to the internet.

This is a switch. Switches are layer 2 devices meaning they change the mac address of the packets they receive and re-send, but they do not effect the IP address.

This represends the internet. For all intensive purposes the internet is any network that is outside our current network.

This represends a DNS server. This server could ether be internal to your network or external. In the scenarios on this website the DNS server will be internal for simplicity.
